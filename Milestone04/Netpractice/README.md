# Netpractice - Network Configuration & Subnetting

Netpractice is a non-coding project at 42 focused on the fundamentals of TCP/IP networking.  
The goal is to solve a series of increasingly complex network configuration puzzles, ensuring that all host computers in a given topology can communicate with each other correctly.

This project demonstrates proficiency in IPv4 addressing, Subnet Masks, and Routing Tables.
